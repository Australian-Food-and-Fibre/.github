# Australian Food and Fibre

Private organisation repositories for Australian Food and Fibre's data platform and reporting infrastructure.

## Repositories

| Repository | Description |
|------------|-------------|
| [data-platform](https://github.com/Australian-Food-and-Fibre/data-platform) | Microsoft Fabric data platform — ELT framework, Power BI reports, and Azure infrastructure |

## Technology Stack

- **Microsoft Fabric** — Medallion architecture (Bronze/Silver/Gold) with OneLake
- **Azure** — Fabric capacities, Automation accounts, Key Vault
- **Power BI** — Semantic models (TMDL) and reports (PBIR)
- **ELT Framework** — Metadata-driven orchestration via ControlDB
- **Integrations** — Fivetran (NetSuite), ICA data processing
